Hackathon

Narcant: A project created at Pitt Challenge 2017. Narcant is an android app to assist in the recooperation of narcotics addicts, by featuring an alarm function between doses (determined by user input), a symptoms recorder if the user needs to report any, and a graphing function for the doctor to analyze the data. Written using Android Studio (Java). NOTE: graphing function is not usable.
